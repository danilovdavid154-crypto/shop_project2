# Shop Project

Интернет-магазин на Django с каталогом, корзиной, регистрацией, оформлением заказа и страницей "Мои заказы".

## Запуск проекта

1. Создайте и активируйте виртуальное окружение:
   `python -m venv venv`
   `venv\Scripts\activate`
2. Установите зависимости:
   `pip install -r requirements.txt`
3. Примените миграции:
   `python manage.py migrate`
4. Заполните проект демо-данными:
   `python manage.py seed_demo_data`
5. Запустите сервер:
   `python manage.py runserver`

## Админка

Создать суперпользователя:

`python manage.py createsuperuser`

Админка будет доступна по адресу `/admin/`.

## Демо-пользователь

После запуска `python manage.py seed_demo_data` создается тестовый пользователь:

- Логин: `demo_user`
- Пароль: `demo12345`

## Что подготовлено для демонстрации

- 5 категорий
- 20 демо-товаров
- 6+ товаров с фото
- минимум 2 товара с `stock=0`
- оформление заказа для гостя и авторизованного пользователя
- страница "Мои заказы" со статусами
